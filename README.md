COVID-19 Data Analysis Project

Overview

This project explores global COVID-19 data using Python to identify trends, patterns, and relationships between infection rates, mortality, vaccination coverage, and development indicators.

The project includes data cleaning, exploratory data analysis (EDA), statistical analysis, time-series analysis, and data visualization.

⸻

Objectives

* Analyze global COVID-19 trends.
* Compare mortality and infection rates across countries and continents.
* Investigate vaccination patterns.
* Explore relationships between development indicators and COVID-19 outcomes.
* Generate actionable insights from public health data.

⸻

Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

⸻

Key Analyses

Data Cleaning

* Removed non-country records.
* Handled missing values.
* Corrected invalid cumulative values.
* Converted date fields to datetime format.

Exploratory Data Analysis

* Cases and deaths distribution.
* Vaccination analysis.
* Country-level comparisons.

Time-Series Analysis

* COVID-19 trends across continents.
* 7-day moving averages.
* Reproduction rate analysis.

Statistical Analysis

* Death-to-case ratio comparison.
* HDI and cardiovascular mortality impact.
* Cross-country comparisons.

⸻

Key Findings

* High-HDI countries generally showed lower COVID-19 fatality rates.
* Vaccination coverage varied significantly across countries.
* Africa showed a higher reported death-to-case ratio despite lower recorded cases.
* Europe experienced multiple large COVID-19 waves during the pandemic.
